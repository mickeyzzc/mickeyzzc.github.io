系列第二篇用 edge-tts 出了配音，第五篇梳理了 TTS 技术发展史。本篇是收官篇——当你的项目不满足于 edge-tts 时，市面上 31 款 TTS 引擎和服务该怎么选？

我们把选型路径分成三层：免费开源引擎（本地运行、零授权费但有 GPU 部署成本）、免费云服务（开箱即用但有额度/限流约束）、付费服务（开箱即用 + SLA + 高级能力）。这并非严格的层级递进——开源引擎搭配自建 GPU 可能比付费云服务长期更便宜，而付费云服务的开箱便利性又是自部署无法比拟的。选择取决于你的预算、人力、时间三者的权衡。

重点引擎详解

CosyVoice（阿里巴巴） — 当前最推荐的本地部署中文 TTS。Apache-2.0 协议完全可商用，支持中文、英文、日文、韩文和粤/川/沪/津/武汉五种方言。零样本克隆仅需 3 秒参考音频，首包延迟低至 150ms，并提供 WebUI、FastAPI、gRPC、Docker、vLLM 等完整服务化路线。支持 14 种细粒度控制标签（[laughter]、[breath] 等），GitHub 获 22k+ stars，社区活跃。

F5-TTS（上海交通大学） — 研究级克隆质量。Flow Matching + DiT 架构，非自回归生成，RTF 约 0.15，完全消除音素对齐需求。代码 MIT 协议，模型权重因 Emilia 训练集为 CC-BY-NC（不可商用）。GitHub 14.5k+ stars，质量在开源社区评价很高。

ChatTTS — 对话场景专用。约十万小时中英文数据训练，专为对话场景优化，支持笑声、停顿等细粒度韵律控制。VRAM 最低约 4GB，RTF 0.3（RTX 4090）。GitHub 39.3k+ stars，注意 AGPL 协议不可商用。

Coqui XTTSv2 — 语言覆盖最广（17 种），零样本克隆仅需约 6 秒参考音频，24kHz 采样率。注意 Coqui 公司已于 2024 年 1 月关闭，由社区继续维护（GitHub 约 45k stars）。

Piper — 边缘设备首选。VITS 架构 + ONNX 导出，CPU 优先，可在树莓派 4 上运行，VRAM < 1GB，支持数十种语言预训练语音。适合 IoT 和嵌入式场景。

Edge TTS 是免费方案中最实用的选择之一。安装 pip install edge-tts 即可使用，无需任何 API Key，基于微软 Neural 网络语音引擎，覆盖 100+ 语言和 400+ 语音，音质优秀。它在第二篇中作为系列配音实战的核心工具——gen_all_voices.py 脚本在 5 次重试 + 指数退避的保护下，能可靠完成数百条音频的批量生成。

但 Edge TTS 并非银弹。它是非官方 API，可能遇到 WebSocket 403 错误（微软限流机制），不适合生产级高并发。如果需要在 Web 产品中集成 TTS，或者需要更稳定的服务，请升级到 Azure Speech Service 正式 SDK（付费 + SLA 保障）。

付费 TTS 服务全面对比

当免费方案不满足时——需要声音克隆、生产级 SLA、广播级音质、中国方言覆盖——就该付费服务上场了。付费 TTS 市场按定价模式分为三类：按字符计费的云 API（国际云厂商）、按月订阅的专业语音公司、按万字符计费的中国云服务。

标准层 $4/百万字符几乎成为行业均价，差异体现在高级层的质量选择和生态系统。

Amazon Polly 提供 30+ 语言和 90+ 声音。Generative voices 和 Long-Form voices（支持最大 20 万字符异步合成）适合长文本有声内容。支持 Speech Marks 输出唇同步时间戳，适用于口型动画场景。2025 年 7 月起新 AWS 用户可获得最高 $200 免费额度。

Azure Speech 语言覆盖最广——140+ 语言/方言、500+ 声音。Custom Neural Voice 可训练定制品牌专属声音（需最少 300 句录音）。情感 SSML 扩展、Viseme 输出（面部动画同步）和 Audio Content Creation Studio（浏览器可视化调优工具）是企业级需求的扎实底座。

Google Cloud TTS 模型选择最为丰富：Neural2、Chirp 3 HD、Journey、Studio Voice 等。免费额度按月重置而非限时一年，小流量项目可长期免费使用标准层。

OpenAI TTS 以极简 API 设计著称，与 GPT-4o、Whisper、Assistants API 无缝集成。gpt-4o-mini-tts 支持自然语言指令控制语音风格，Realtime API 支持端到端亚秒级语音对话。价格为 $15/百万字符起，约为 ElevenLabs 的 1/10，但声音数量（6+）和音色选择有限。

ElevenLabs 是语音质量最高的服务，MOS 约 4.3 达到广播级水准。70+ 语言、11,000+ 声音库，Flash v2.5 延迟约 75ms。声音克隆是它的王牌——Starter 以上层级支持即时克隆（30 秒音频），Creator 以上支持专业克隆（30 分钟训练音频）。Dubbing Studio 支持 70+ 语言的配音 + 唇同步，ElevenAgents 提供 STT+LLM+TTS+电话一体化服务。还支持 Sound Effects 和 Music 生成。

国内 TTS 市场以云厂商为主，价格竞争激烈，基础层几乎统一定价在 0.3-0.4 元/万字符。差异体现在大模型层、声音克隆、方言覆盖和免费额度上。

阿里云 基于 Sambert 模型和开源的 CosyVoice 大模型，侧重电商场景音色（客服音等），新用户 3 个月 100 万字符/月免费。如果你用阿里云 ECS 部署，集成最方便。

百度智能云 支持方言（四川话、粤语）和情感语音（开心/悲伤），精品版 24k 音质。免费额度充足（1 年 200 万字符/月），是免费优先场景的不错选择。

火山引擎（字节跳动/豆包） 基于 Seed-TTS 的豆包语音合成模型 2.0，提供端到端超拟人对话的实时语音模型。声音复刻 2.0 和情绪精细化控制是特色能力，定价 5 元/万字符。适合字节生态（抖音、飞书）内集成的团队。

最新应用场景

TTS 正在从一个工具性能力演进为语音交互的底层基础设施。以下是当前最活跃的 8 个应用方向：

实时语音对话（GPT-4o Voice、豆包实时语音、Sesame CSM）是 2024-2025 年最热门的应用方向，与传统 STT+LLM+TTS 三段式管道不同，新一代系统直接在语音模态推理，实现亚秒级响应和情绪自适应。

影视配音与内容本地化（ElevenLabs Dubbing Studio）可自动翻译并配音到 70+ 语言，支持唇同步，保留原始情感和语调，大幅降低传统配音成本。

有声内容制作（有声书、播客、新闻播报）已高度自动化，支持多角色/多声音切换、情感控制和 Studio Sound 音质增强。

无障碍辅助（Speechify、Apple VoiceOver）为视障人士朗读屏幕内容，多平台支持，自然度提升显著改善体验。

智能客服与 IVR 深度融合情感识别与情感化回应，Custom Neural Voice 定制品牌专属声音，Viseme 输出支持虚拟人面部动画。

游戏与虚拟人 通过动态语音生成和 Viseme 面部同步驱动 NPC 和数字人。

教育与语言学习 朗读课文、发音示范、个性化语音导师，支持口语评测。

嵌入式与 IoT（科大讯飞离线 SDK、Piper）在智能音箱、车载系统、树莓派上离线运行 TTS，保障隐私和低功耗。

付费 TTS 的定价模式直接影响选型决策。按字符计费（AWS Polly、Azure、Google Cloud、OpenAI、中国云）适合用量不确定的按需场景，无固定成本但随时可伸缩。订阅制按月（ElevenLabs、Play.ht、Murf.ai、WellSaid）适合稳定用量的内容创作团队——每月产出固定时长的配音内容，订阅费封顶不会因月间波动产生意外支出。按分钟计费（ElevenLabs ElevenAgents 的 $0.08/分钟）用于实时对话/电话场景，计费粒度与使用维度匹配。分层订阅（IBM Watson Lite/Plus/Premium）以阶梯功能划分适用不同规模企业。此外还有科大讯飞的永久免费额度（10 万字符/月），是小规模长期使用的良心选择——不经任何付费行为即可一直使用，但受限于功能和服务级别。
